Class TValue

一个通用数据类型，用来存放整数、浮点数、字符串和其它对象。 在C/C++中，一般不需动态创建对象，直接声明并初始化即可。如：

value_t v;
value_set_int(&v, 100);

在脚本语言中，需要动态创建对象。
