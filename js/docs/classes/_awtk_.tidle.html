Class TIdle

idle可以看作是duration为0的定时器。 idle可以用来实现一些异步处理。 示例：

static ret_t something_on_idle(const idle_info_t* info) {
widget_t* widget = WIDGET(info->ctx);
edit_t* edit = EDIT(widget);
...
return RET_REMOVE;
}
...
idle_add(something_on_idle, edit);

在非GUI线程请用idle_queue。
